BigNum is designed to wrap a simple struct containing a vector and offer the user to create these objects just as they would a standard type, but BigNum allows operations on an arbitrary size and is not limited to a limit.

Examples

use bignum::BigNum;
use bignum::inits::{One, Zero};

let a = BigNum::from_u32(5);    // 5
let b = &a + &One::one();       // 6

let c = &a + &b;                // 11
let d = &c - &b;                // 5
let e = &c * &d;                // 55
let h = &e / 5;                 // 11

Operators are purposely implemented on references to objects. although copyable, performing arithmatic via copies would be very inefficient compared to this implementation.
